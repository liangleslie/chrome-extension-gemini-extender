# Gemini Enterprise Extended

Enhance your Google Gemini web experience with this feature-rich, Manifest V3-compliant Chrome Extension.

This extension bridges the gap between an isolated side panel UI and the live Gemini webpage DOM, providing advanced controls, custom navigation menu injection, and structured prompt building.

---

## 🏗️ Architecture Overview

The extension utilizes a modular messaging system to send information securely across execution contexts:

```
[ Side Panel UI ] <---> [ Background Service Worker ] <---> [ Content Script ] <---> [ Gemini DOM ]
 (Prompt Builder)             (Message Router)                 (DOM Injector)
```

---

## ✨ Key Features

- **Menu Restoration**: Injects missing or customized navigation elements directly into Gemini's native sidebar.
- **Sidebar Workspace (Side Panel)**: A dedicated workspace panel within Chrome to draft complex prompts, manage prompt history, and interact with the main page.
- **Extract-Clean-Validate Loop**: An robust message parsing layer that sanitizes, parses, and validates JSON structures generated by Gemini, returning raw data payloads directly to the side panel UI.
- **DOM Selector Mapping**: Uses a centralized mapper to adapt dynamically to class name or layout changes on `gemini.google.com`.

## 🚀 Installation & Getting Started

Since this extension is in active development, you can load it locally as an unpacked developer extension:

1. **Download/Clone** this repository to your local machine.
2. Open Google Chrome and navigate to: `chrome://extensions/`.
3. Enable **Developer mode** using the toggle switch in the upper-right corner.
4. Click **Load unpacked** in the top-left corner.
5. Select the project directory (`chrome-extension-gemini-extender`).
6. Pin the extension for quick access.

---

## 💡 Usage

1. Open your browser and navigate to [gemini.google.com](https://gemini.google.com/).
2. Click the extension icon in your toolbar, or open Chrome's Side Panel dropdown and choose **Gemini Enterprise Extended**.
3. Use the Sidebar Workspace to construct your prompt template, send it, and watch the extension parse structured responses dynamically!
